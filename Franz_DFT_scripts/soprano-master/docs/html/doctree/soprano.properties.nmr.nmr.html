class soprano.properties.nmr.nmr.MSDiagonal(name=None, **params)¶

Bases: soprano.properties.atomsproperty.AtomsProperty

MSIsotropy

Produces an array containing eigenvalues and eigenvectors for the symmetric part of each magnetic shielding tensor in the system. By default saves them as two new arrays as well. Requires the Atoms object to have been loaded from a .magres file containing the relevant information.

Parameters:
save_info (bool): if True, save the diagonalised tensors in the
Atoms object’s info. By default True.
Returns:
ms_diag (np.ndarray): list of eigenvalues and eigenvectors
